## 1️⃣ What is the difference between var, let, and const?

- var is global scope and can be reassign anywhere and its hoisted with initialization undefined.
- let is a block scope which can be reassign it's also hoisted but it creates a temporary dead zone which is why it's can't call without declaration first and it'll throw a Reference Error.
- const is a block scope as well but it can't be reassign later and it's hoisted but and it also creates a temporary dead zone where hoisting stops.

## 2️⃣ What is the spread operator (...)?

- spread operator is a js feature which can be use for copy,patch or pass data in array and object and it used for expands arrays, strings, object.

## 3️⃣ What is the difference between map(), filter(), and forEach()?

- map() method is use to creates a new array and it dose't modify original array. It is generally used for creating new array without effecting original one.
- filter() method is use to filter something with conditions. It returns a new array based on the condition provided. It also doesn't affect the original array.
- forEach() method is to iterate each elements of an array. forEach is use to update variables or modifying element.

## 4️⃣ What is an arrow function?

- Arrow function is an compact function which is alternative of a traditional function expression.

## 5️⃣ What are template literals?

- Template literals are a feature of ES6 which allow to dynamically manipulates strings and can pass expressions including multi-line of things using the backticks `` instead of single or double quotes.
